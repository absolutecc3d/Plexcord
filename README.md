# Plexcord website

A static, responsive landing page for Plexcord.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Before publishing

Replace every occurrence of:

```text
YOUR_USERNAME
```

with the GitHub username that owns the Plexcord repository.

## Run locally

Open `index.html` directly in a browser, or start a basic local server:

```powershell
npx.cmd serve .
```

## Publish

The site can be hosted on GitHub Pages, Netlify, Cloudflare Pages, Neocities, or any standard static host.
