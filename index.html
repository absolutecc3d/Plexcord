Installation

Build it into Vencord.

Custom plugins are placed in Vencord's userplugins directory and included when Vencord is built.

1

Clone Vencord

git clone https://github.com/Vendicated/Vencord.git
cd Vencord
2

Install dependencies

pnpm.cmd install --frozen-lockfile
3

Add Plexcord

New-Item -ItemType Directory -Force -Path "src/userplugins" | Out-Null
git clone https://github.com/absolutecc3d/Plexcord.git src/userplugins/Plexcord
4

Build and inject

pnpm.cmd build
pnpm.cmd inject
5

Enable the plugin

Restart Discord, open Settings → Vencord → Plugins, search for Plexcord, and enable it.

No pretending

Plexcord changes the view, not the account.

A custom username is not a real username change. A custom badge is not an official Discord badge. Plexcord is a presentation layer for people who want more control over how their client looks.

Keep that distinction clear when sharing screenshots or distributing the plugin.

FAQ

Things worth knowing.

Will other Discord users see my changes?

Not by default. The changes are rendered locally by Plexcord.

Does Plexcord change my real Discord username?

No. It only changes the username displayed by the client.

Does Plexcord grant official badges?

No. Custom badges are visual additions and do not exist on your Discord account.

Why does installation require a Vencord source build?

Plexcord is a custom user plugin, so it must be included in the Vencord source tree before Vencord is built.
